# 🚀 Gemini DOM Pruner 🚀

**Легковесное браузерное расширение для устранения лагов и фризов в длинных сессиях Google Gemini.**

> [!IMPORTANT]
>
> ### ОГРАНИЧЕНИЯ РАСШИРЕНИЯ
> Данный фикс спасает **исключительно в длинных переписках** (от 15-20 сообщений), когда браузер начинает захлебываться от количества элементов на странице. 
> Если у вас тормозит интерфейс на пустых чатах (2-3 сообщения), то это расширение не поможет, проблема кроется в другом.

## ℹ️ Как это работает

При долгом общении Gemini сохраняет все старые сообщения в DOM-дереве. При каждом новом ответе нейросети или вашем вводе текста, браузер пытается пересчитать (reflow/repaint) вообще всю страницу. На длинных чатах это приводит к сильным лагам, фризам при вставке текста и дерганому скроллу.

Расширение инжектит крошечный нативный CSS-код, применяющий свойство `content-visibility: auto` к контейнерам сообщений Gemini. 
В следствие чего, браузер рендерит только те элементы, которые находятся на вашем экране. Всё, что осталось за пределами видимости, замораживается и не расходует ресурсы процессора и оперативной памяти.

## ⚙️ Установка

Поскольку расширение работает локально, не содержит фоновых скриптов и не требует разрешений, оно устанавливается напрямую из исходников (в режиме разработчика).

### Для Google Chrome (а также Brave, Vivaldi, Яндекс.Браузер)

1. Скачайте этот репозиторий: зеленая кнопка **Code** -> **Download ZIP**
2. Распакуйте архив в любую удобную папку на компьютере (желательно без кириллицы в пути)
3. В браузере перейдите по адресу: `chrome://extensions/`
4. В правом верхнем углу включите тумблер **«Режим разработчика»** (Developer mode)
5. Нажмите на появившуюся кнопку **«Загрузить распакованное расширение»** (Load unpacked)
6. Выберите папку, в которую вы распаковали архив (где лежит файл `manifest.json`)
7. Обновите вкладку с Gemini (F5), чтобы стили применились.

### Для Microsoft Edge

1. Скачайте и распакуйте ZIP-архив с кодом
2. Введите в адресную строку браузера: `edge://extensions/`
3. В меню слева (или в нижнем левом углу) включите тумблер **«Режим разработчика»**
4. В верхней части страницы нажмите **«Загрузить распакованное расширение»**
5. Укажите путь к папке с файлом `manifest.json`
6. Обновите открытую страницу чата Gemini.
